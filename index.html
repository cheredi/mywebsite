Swali-AI - RAG Interview Coach

Swali-AI is a production-grade RAG system for technical interview preparation. It combines retrieval, progressive hints, and role-aware coaching to help candidates go beyond memorized answers and build deeper understanding.

Business impact: Converts interview prep into a structured coaching workflow with faster guidance and clearer feedback loops.
Features from repo: Smart Q&A, Progressive Hints, Adaptive Difficulty, Similar Problems, and Follow-up Questions.
RAG Embeddings ChromaDB FastAPI React + Vite Gemini API sentence-transformers

  • Data layer: ChromaDB vector store + sentence-transformer embeddings.
